# LoginScriptTo3S

Dies ist ein Loginscript damit ein AP von 3S einen Fehler melden kann. 

Es wird ausgefüht aus dem Pc-Assisten. dabei startet einen Browser der dieses Script ausführt. 

Die Nutzung des Script ist nicht momentan Möglich, da keine Datenbankanbidung im script vorhanden ist.

Das HTML-Design ist mit bootstrap erstellt. Alle anderen Code wurde durch hilfe von: 
  
dem Tutorial : https://www.php-einfach.de/experte/php-codebeispiele/loginscript/

und Das Mailing Verfahren ist aus dem Githubu: https://github.com/PHPMailer/PHPMailer/

diese beide hilfreiche Gits haben ausgenutzt und daraus einen passendes Verfahren was uns als Unterstützung dient.
